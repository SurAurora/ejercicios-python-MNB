# Ejercicios Python - Unidad B

Guia de ejercicios de Python para la Unidad B del curso Modelamiento Numerico
Basico.

## Estructura

- `clase_01_programacion_basica.md`: guia de ejercicios para variables,
  condicionales, bucles, funciones y pseudocodigo.
- `clase_01_programacion_basica.py`: archivo base para completar los ejercicios
  de la primera clase.
- `clase_02_series_tiempo.md`: guia de ejercicios para analisis de series de
  tiempo atmosfericas.
- `clase_02_series_tiempo.py`: archivo base para completar los ejercicios de la
  segunda clase.
- `datos_clase_02_series.csv`: datos sinteticos para la segunda clase.

## Como usar este material

1. Leer la guia `.md` de la clase correspondiente.
2. Abrir el archivo `.py` asociado.
3. Completar los espacios marcados con `TODO`.
4. Ejecutar los scripts desde esta carpeta:

```bash
python clase_01_programacion_basica.py
python clase_02_series_tiempo.py
```

Para la clase 2 se recomienda tener instaladas las librerias:

```bash
pip install numpy pandas matplotlib
```
