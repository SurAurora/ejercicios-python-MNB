# Ejercicios Python - Unidad B

Material preparado a partir del calendario de actividades de Semana 2 del curso
Modelamiento Numerico Basico.

## Estructura

- `clase_01_programacion_basica.md`: guia de ejercicios para variables,
  condicionales, bucles, funciones y pseudocodigo.
- `clase_01_programacion_basica.py`: plantilla para estudiantes.
- `clase_02_series_tiempo.md`: guia de ejercicios para analisis de series de
  tiempo atmosfericas.
- `clase_02_series_tiempo.py`: plantilla para estudiantes.
- `datos_clase_02_series.csv`: datos sinteticos para la segunda clase.
- `soluciones/`: codigos resueltos para uso docente.

## Uso sugerido

1. Entregar a estudiantes los archivos `.md`, las plantillas `.py` y el `.csv`.
2. Mantener la carpeta `soluciones/` solo para preparacion de clase o pauta.
3. Ejecutar los scripts desde esta carpeta:

```bash
python clase_01_programacion_basica.py
python clase_02_series_tiempo.py
```

Para la clase 2 se recomienda tener instaladas las librerias:

```bash
pip install numpy pandas matplotlib
```
